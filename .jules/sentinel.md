## 2026-02-07 - Testing Nuxt Server Handlers\n**Vulnerability:** Difficulty in testing server handlers due to auto-imports.\n**Learning:** Nuxt server handlers rely on auto-imports (`defineEventHandler`, `readMultipartFormData`, `createError`) which are not present in standard Vitest unit tests.\n**Prevention:** Always mock these globals or use `@nuxt/test-utils` environment when writing unit tests for server API handlers.
